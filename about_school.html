Human Computer Interaction

HCI was all about determining a need, creating a low-fidelity product, getting user feedback and then using that feedback to create a high-fidelity product. I really enjoyed this class and felt like the class offered the most growth. I think this is primarily because user testing and feedback is not something we do a lot in my role as a designer at work. Much of the feedback we receive is from our learners by way of an L1 survey and we rely on the learner's performance metrics to tell us how effective the course was at teaching the learner. After this class, I brought this up to my team and suggested that we implement a process that allows for more testing and feedback because it is something I'm excited about.
